# Setup Checklist

Follow these steps to get the project fully configured:

- [ ] **Initialize Git**: Run `git init` in the root directory.
- [ ] **Install Dependencies**: Run `npm install`.
- [ ] **Configure Agents**: Create an `AGENTS.md` file to define the project's AI agents.
- [ ] **Environment Setup**: Ensure Node.js is installed and configured.
- [ ] **Verify Application**: Run `npm start` to ensure the server starts correctly.
